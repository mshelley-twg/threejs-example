# threejs-example
Just learning about three.js!

## Running Locally
* Clone this repo
* `npm install`
* `npm run start`
* Open `http://localhost:8080/`

## Controls
| Key | Description                         |
| --- | ----------------------------------- |
| W   | Move player forward                 |
| A   | Move player left around the camera  |
| S   | Move player backward                |
| D   | Move player right around the camera |
